JavaScript Primitives and Objects

In JavaScript, data types are categorized into two main groups:

  • Primitive Types: Basic data types that hold a single value.
  • Objects: Complex data structures that can store collections of data and functionality.

Primitive Types

Primitive types are immutable and represent simple values. They include:

Type Description Example
String Sequence of characters. "Hello, World!"
Number Numeric values, including integers and floating-point numbers. 42, 3.14
BigInt Represents integers larger than Number.MAX_SAFE_INTEGER. 12345678901234567890n
Boolean Represents logical values: true or false. true, false
Undefined Indicates a variable that has been declared but not assigned a value. let x; // x is undefined
Null Represents an intentional absence of value. let y = null;
Symbol Unique and immutable value, often used as object property keys. Symbol("id")

Examples of Primitives

// String
let greeting = "Hello, World!";
console.log(greeting); // Output: Hello, World!

// Number
let age = 25;
console.log(age); // Output: 25

// BigInt
let largeNumber = 12345678901234567890n;
console.log(largeNumber); // Output: 12345678901234567890n

// Boolean
let isJavaScriptFun = true;
console.log(isJavaScriptFun); // Output: true

// Undefined
let notAssigned;
console.log(notAssigned); // Output: undefined

// Null
let emptyValue = null;
console.log(emptyValue); // Output: null

// Symbol
let uniqueKey = Symbol("key");
console.log(uniqueKey); // Output: Symbol(key)

Objects

Objects are collections of key-value pairs. They are mutable and can store multiple values and functions.

Types of Objects

  • Object Literals: Simplest way to create objects.
  • Arrays: Ordered list of values.
  • Functions: Callable objects with reusable code.
  • Date: Represents date and time.
  • Other Built-in Objects: Examples include Math, RegExp, and Set.

Examples of Objects

// Object Literal
let person = {
    name: "Alice",
    age: 30,
    greet: function() {
        console.log("Hello, " + this.name);
    }
};
console.log(person.name); // Output: Alice
person.greet(); // Output: Hello, Alice

// Array
let colors = ["red", "green", "blue"];
console.log(colors[1]); // Output: green

// Function
function add(a, b) {
    return a + b;
}
console.log(add(5, 3)); // Output: 8

// Date
let today = new Date();
console.log(today.toDateString()); // Output: Current date in readable format

// Set
let uniqueNumbers = new Set([1, 2, 3, 3]);
console.log(uniqueNumbers); // Output: Set { 1, 2, 3 }
